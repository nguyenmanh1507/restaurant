# Restaurant One Page Template

## Tool use
* Yeoman for scaffolding project
* Bower for manage dependencies
* Gruntjs for automatic tasks

## Dependencies
* Zurb Foundation 5
* FontAwesome
* jQuery
* Datetimepicker
* headroom
* vivus
* jscrollpane (option)
* stellar
